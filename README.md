# Beamer Theme Technikum

Custom Beamer theme for Fachhochschule Technikum Wien (FHTW).

## Structure

The theme follows Beamer's modular design pattern:

- `beamerthemetechnikum.sty` - Main theme file
- `beamercolorthemetechnikum.sty` - Color definitions
- `beamerfontthemetechnikum.sty` - Font definitions  
- `beamerinnerthemetechnikum.sty` - Inner elements (lists, blocks, theorems)
- `beamerouterthemetechnikum.sty` - Outer elements (header, footer, frametitle)

## Usage

In your Beamer document:

```latex
\documentclass{beamer}
\usetheme{technikum}
```

## Development

Theme files are created iteratively as needed for lecture slides.
Files will be added as design requirements emerge.
